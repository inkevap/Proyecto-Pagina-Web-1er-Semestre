## 🌐 Páginas Incluidas

### 1. Main.html

Página principal que sirve como punto de entrada al proyecto. Contiene enlaces a las tres secciones principales:

- Perfil Personal y Profesional (CV)
- Retroalimentación del curso
- Crítica constructiva
  
  ### 2. CV.html
  
  Currículum Vitae profesional con:
- Información de contacto
- Perfil profesional
- Experiencia laboral
- Educación
- Habilidades técnicas (Java, Spring Boot, Microservicios, etc.)
  
  ### 3. Curso.html
  
  Retroalimentación del curso de Introducción a los Sistemas de Cómputo.
  
  ### 4. Retroalimentacion.html
  
  Crítica constructiva dirigida al Ingeniero Harold Cancinos.
  
  ## 🛠️ Tecnologías Utilizadas
- **HTML5** - Estructura semántica

- **Meta tags SEO** - Optimización para motores de búsqueda
  
  ## 🚀 Cómo Visualizar el Proyecto
1. Clona o descarga este repositorio
2. Abre el archivo `Main.html` en tu navegador web preferido
3. Navega por las diferentes páginas usando los enlaces proporcionados
   
   ```bash
   # Opción 1: Abrir directamente
   xdg-open Main.html  # Linux
   open Main.html      # macOS
   start Main.html     # Windows
   # Opción 2: Usar un servidor local
   python3 -m http.server 8000
   # Luego visita http://localhost:8000/Main.html
   ```
